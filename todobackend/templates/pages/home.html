Welcome To ToDo API Playground

  • There is only one API that provides full RESTful service on ToDo Items. It's designed to help everyone to test their code to integrate with a RESTful API.
  • ToDo items are grouped by sessions, you can override the session id by specifying SESSION-ID value in your request header.
  • There is a maximum rows in ToDo table globally to protect the service not being overrun, once maximum rows has reached, it will start evicting oldest items.

Installation
--------------

Setting Up Dev Environment
^^^^^^^^^^^^^^^^^^^^^^^^^^

1. install docker
2. git clone the repo
3. `make init` to create the initial database

Running the development server
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
::

   docker-compose up -d runserver

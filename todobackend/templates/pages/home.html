Welcome To ToDo API Playground

  • There is only one API here, which provides full RESTful service on ToDo Items. Purposed to help everyone to test their code to integrate with a RESTful API.
  • ToDo items are grouped by sessions, you can override the session id by specifying SESSION-ID value in your request header.
  • There is a maximum rows limit in ToDo table globally to protect the service from being overrun, once maximum rows has reached, it will start evicting oldest items.
